#Análisis Paralelo de Datos Financieros  
Procesamiento eficiente de series temporales mediante división recursiva y ejecución en paralelo

Este proyecto implementa un sistema capaz de analizar grandes conjuntos de datos financieros mediante un enfoque *divide and conquer* (dividir y conquistar) combinado con procesamiento en paralelo. Esto permite mejorar significativamente el rendimiento al trabajar con series de tiempo extensas, obteniendo métricas estadísticas y financieras de forma eficiente, consistente y escalable.



#Objetivo del Proyecto

El objetivo principal es ofrecer una herramienta que:

- Procese grandes series temporales de precios.
- Divida la serie en segmentos usando un método recursivo.
- Procese cada segmento simultáneamente mediante paralelismo.
- Calcule métricas estadísticas y financieras fundamentales.
- Combine los resultados parciales para obtener indicadores globales precisos.

Este enfoque está diseñado para tareas como:

- Análisis financiero masivo  
- Sistemas de trading  
- Análisis de riesgo  
- Computación distribuida  
- Estudio de comportamiento histórico de precios
